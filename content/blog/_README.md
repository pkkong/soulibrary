블로그 글 추가 방법

1. 이 폴더에 `영문-slug.md` 파일을 추가합니다.
2. 아래 형식을 복사해서 작성합니다.
3. 저장 후 `/blog`에서 자동으로 보이는지 확인합니다.

예시:

---
title: 글 제목
description: 목록에 보이는 짧은 설명
category: 가이드
date: 2026-05-21
---

## 소제목

본문을 짧게 씁니다.

- 목록 항목 1
- 목록 항목 2

![이미지 설명](/static/img/blog/example.png)

지원하는 문법은 `##`, `###`, 문단, `-` 목록, 외부 링크, `/static/` 이미지 정도입니다.
