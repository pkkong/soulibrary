블로그 글 추가 방법

1. 이 폴더에 `영문-slug.md` 파일을 추가합니다.
2. 아래 형식을 복사해서 작성합니다.
3. 저장 후 `/blog`에서 자동으로 보이는지 확인합니다.

예시:

---
title: 글 제목
description: 목록에 보이는 짧은 설명
category: 가이드
date: 2026-05-21
---

## 소제목

본문을 짧게 씁니다.

- 목록 항목 1
- 목록 항목 2

![이미지 설명](/static/img/blog/example.png)

지원하는 문법은 `##`, `###`, 문단, `-` 목록, 외부 링크, `/static/` 이미지 정도입니다.

목차 링크가 필요하면 제목 뒤에 `{#영문-id}`를 붙입니다.

## 검색하기 {#guide-search}

편집 기준:

- 처음 이용자가 실제로 따라 할 수 있는 순서로 씁니다.
- Soulib이 해주는 일과 각 도서관, 공급사, 공식 앱에서 해야 하는 일을 구분합니다.
- 앱 설치 링크, 회원증 안내, 도서관 정책은 공식 사이트나 공식 앱스토어 링크만 사용합니다.
- 서울온은 서울 도서관 회원증과 시설카드 관리를 시작하는 추천 출발점으로 설명하되, 모든 도서관 전자책 이용이 자동으로 열리는 것처럼 쓰지 않습니다.
- 화면 설명이 필요한 글은 실제 스크린샷이나 설명 이미지를 함께 준비합니다. 임시 이미지나 공개 TODO 문구는 본문에 넣지 않습니다.

우선 개선할 글:

- `seoul-on-library-guide.md`: 서울온 설치, 가입, 시설카드 추가, 도서관별 인증 caveat, 앱 설치 순서를 가장 먼저 보강합니다.
- `soulib-guide.md`: 이미 보유한 Soulib 화면 스크린샷을 기준으로 검색, 상세, 내 서재, 오류 신고 흐름을 유지 보강합니다.
- `ebook-library-status-guide.md`: 대출 상태 용어를 실제 행동으로 연결하고, 상세 화면 상태 예시 이미지를 추가하면 좋습니다.
